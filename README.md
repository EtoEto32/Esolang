## Overview
Urban Müllerがコンパイラがなるべく小さくなる言語として考案した。なんとそのサイズ123バイト、インタプリタは98バイトである。
難解プログラミング言語で最も有名と言っていいんじゃないかと言われている"Brain fuck"をちょろっと触ってみる。因みに開発者のUrban Müller（アーバンミュラー）は現在、スイスの検索エンジンであるsearch.chで働いているそう。
